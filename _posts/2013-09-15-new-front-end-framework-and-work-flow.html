Things are happening fast these days in our industry.  If you have been using Yeoman, you know that front end development is no longer editing a few text files and FTPing them to a server.

Todd Motto ( toddmotto.com) has introduced a new framework and workflow package that looks to be a nice addition to any dev's arsenal.  FireShell is a nice collection of javascript, sass/css and html templates as well as a grunt file to manage your workflow with Grunt.
