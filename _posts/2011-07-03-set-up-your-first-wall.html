In this lesson, you will set up your first wall with VitalWall.

 

Log in to your account

Enter your email and password to go to your accounts main page.

Click New Wall

Click "New Wall"

Fill In Wall Name

Fill in the name of the Wall. This will be used in the <title> tag when viewing your wall.

Select Wall Type

Select Wall Type. (More options coming very soon)

Enter Link Text

Enter text to be used in the link to your wall.

Click Add Wall

Click "Add Wall"

Copy Code to insert into your page

Copy the code to your clipboard. This is the code you will insert into your web page where you want to track data.

Insert Code Into Your Page

Use your favorite web page or text editor and load the page you want to track. Insert the code on your clipboard to your code before the end </body> tag in your page.

Change the 'DATA TO PASS TO WALL' to your data.

This is the most important step. Change the String 'DATA TO PASS TO WALL' to whatever you want to pass to the wall. Make sure you leave the quotes. And that's it. Your wall will start collecting data.
